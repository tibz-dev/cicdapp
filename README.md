# Spring Boot CI/CD Pipeline

## Overview

This project demonstrates a complete DevOps CI/CD pipeline for deploying a containerized Spring Boot REST API using modern DevOps tools and best practices.

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Docker
- Docker Compose
- Jenkins
- GitHub
- AWS EC2

## Features

- RESTful API
- MySQL database integration
- Dockerized application
- Multi-container deployment using Docker Compose
- Jenkins CI/CD pipeline
- GitHub integration
- Automated deployment
- AWS-ready architecture
